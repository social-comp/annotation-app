# Annotation-App



## Description
node js app for annotating categories in weibo social media posts



### Dependencies


    "body-parser": "^1.19.0",
    "cookie-parser": "~1.4.4",
    "debug": "~2.6.9",
    "ejs": "^2.7.4",
    "express": "^4.16.4",
    "express-rate-limit": "^5.3.0",
    "helmet": "^4.6.0",
    "http-errors": "~1.6.3",
    "jquery": "^3.6.0",
    "morgan": "~1.9.1",
    "sqlite3": "^5.0.2"

### Installing

```
npm install jquery express ejs body-parser cookie-parser debug ejs helmet http-errors morgan sqlite3

```


### Executing program

* How to run and start the app

```
npm start
```
http://localhost:3000/
