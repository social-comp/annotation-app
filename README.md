# Annotation-App



## Description
node js app for annotating categories in weibo social media posts



### Dependencies

* ejs
* express
* body-parser
* jquery

### Installing

```
npm install jquery express ejs body-parser

```


### Executing program

* How to run and start the app

```
npm start
```
http://localhost:3000/
